# Braille Translator
## Role
Model training and evaluation - Created telegram bot
This project aims to bridge the gap between Braille literacy and accessibility through the integration of cutting-edge technologies. Leveraging the power of image processing and deep learning, the system provides a robust solution capable of accurately interpreting Braille characters from scanned images. Furthermore, the project extends beyond mere text extraction by incorporating a text-to-speech conversion step. By synthesizing the extracted English text into audible speech, the system empowers individuals with visual impairments to access Braille content effortlessly.

# DataVerse ChatBot
A versatile Python-based AI chatbot leveraging Retrieval-Augmented Generation (RAG) for real-time conversations. It extracts data from web pages and nearly all file formats (PDFs, DOCX, CSV, etc.) using crawl4ai, scrapegraphai, langchain, and docling, storing content in markdown for efficient indexing. Deployable as WhatsApp/Telegram bots or an iframe, it supports voice messages and integrates with LLMs like OpenAI, Claude, Cohere, DeepSeek, Gemini, Grok, and Mistral. Features include web crawling, multi-format data extraction, FAISS-powered vector retrieval, and context-aware responses. It offers chat history persistence in SQLite, uncertain response detection with email alerts, and a monitoring service. An admin dashboard provides usage metrics, content management, and account settings. The system supports dataset creation, model training (92.7% accuracy), and embedding generation with modular, extensible design.
